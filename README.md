Conversor de Moedas em C#

---

### Descrição

Este é um simples aplicativo de console em C# que permite aos usuários converter um valor em reais para ienes (JPY) usando uma taxa de câmbio específica. O programa inclui uma interface amigável para entrada de dados e exibe o resultado com o símbolo do iene.

### Instruções de Uso

1. Abra o terminal no diretório do projeto.
2. Execute o comando `dotnet run`.
3. Insira o valor em reais quando solicitado.
4. Insira a taxa de câmbio para iene (JPY) quando solicitado.
5. O programa calculará e exibirá o valor convertido em iene.

Ao finalizar, o programa solicitará que o usuário pressione Enter para fechar.

### Requisitos

Certifique-se de ter o .NET SDK instalado no seu sistema. Caso contrário, você pode baixá-lo em [dotnet.microsoft.com](https://dotnet.microsoft.com/download).
